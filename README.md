# Campus Swap App

A mobile platform enabling university students to securely swap, buy, or sell books, gadgets, and other items on campus. The app ensures safety, real-time updates, and location-based matching for seamless student-to-student exchanges.

## 🔧 Features
- Secure login via email or student ID (Firebase Auth)
- Post items with descriptions and images
- Browse categorized listings with filters
- Real-time chat between users
- Location-based item prioritization

## 🛠️ Technologies Used
- Android Studio
- Java & XML
- Firebase (Authentication, Firestore, Cloud Messaging, Storage)

## 🚀 How to Run
1. Clone the repository
2. Open in Android Studio
3. Set up Firebase config (google-services.json)
4. Build and run on Android emulator or device
